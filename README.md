# Data-cleaning-using-power-query-
Cleaned a raw sales dataset using Power Query in Power BI by handling missing values, removing duplicates, correcting data types, and preparing the data for analysis.

### Tools Used
- Power BI
- Power Query
- Microsoft Excel

### Data Cleaning Steps
✔ Removed duplicates
✔ Handled missing values
✔ Corrected data types
✔ Standardized text values
✔ Removed invalid records
✔ Verified data quality

### Outcome
The dataset was transformed into a clean, analysis-ready dataset suitable for reporting and dashboard creation.
